# Nannu AI - Medical Assistant

Advanced AI-powered medical assistant with Gemini API integration.

## Features

- 💬 AI Medical Chat Consultation
- 🩻 Medical Image Analysis (X-ray, MRI, CT scans)
- 📋 Symptom Analyzer
- 💊 Drug Interaction Checker
- 🥗 Personalized Nutrition Planner
- 📄 Health Record Analysis (PDF/CSV)
- ❤️ Risk Assessment (Diabetes, Heart, Kidney)

## Setup

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Create `.env` file with your Gemini API key
4. Run: `python app.py`
